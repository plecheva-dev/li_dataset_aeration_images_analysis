## Folder concept

Stand 12/02/2026

* Data: contains raw, processed and result data in the form of images, cvv, text, etc. 
* helper_functions: a small package that can be installed in the virtual environment of the repository to access recurring functions
* notebooks: I use Jupyter interactive console for code development. However, when I have good analysis finished or some codes, figures and text to combine, write a report with jupyter notebooks.
* scripts: python files with inputs at the top (params, paths) and does some calculations and have an output that can be visualized or exported.
